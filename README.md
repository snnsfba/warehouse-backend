#  Warehouse Backend System

Микросервисная бэкенд-система для управления складскими запасами с использованием современных технологий.

##  Технологический стек

### Backend
- **Go** (Golang) - основной язык разработки
- **gRPC/HTTP** - межсервисная коммуникация
- **PostgreSQL** - основное хранилище данных
- **Redis** - кэширование и сессии
- **Apache Kafka** - обработка событий и логов
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация контейнеров

### Архитектура
- **Микросервисная архитектура**
- **API Gateway** паттерн
- **Структурированное логирование** (debug, info, error)
- **Репозиторий на Git** с CI/CD готовностью
